### Hi, I’m Timur 👋

I work on [**Cozystack**](https://github.com/cozystack/cozystack) — a CNCF Sandbox,
open-source framework for building cloud platforms on Kubernetes and KubeVirt.

I don’t ship the core code, but I help shape where the project goes and how it shows
up in the world: proposing features and product direction, shaping the user experience,
and owning the docs, blog, and community around it. The work that sits between the
commits and the people who actually use them.

By trade I sit across product, operations, and engineering — translating between what
users need and what the platform does, and helping more people run cloud-native
infrastructure on free software.
